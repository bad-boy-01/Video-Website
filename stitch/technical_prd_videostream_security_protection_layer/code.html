Technical Specification

Video Delivery Layer

  • Infrastructure: Cloudflare Stream or AWS S3 + CloudFront CDN integration.
  • Protocol: Mandatory HLS (HTTP Live Streaming). Raw MP4 delivery is strictly prohibited to ensure adaptive bitrate and encryption.
  • URL Generation: Server-side Edge Functions must generate short-lived signed URLs.
  • TTL: Expiry set to 1-5 minutes to prevent URL sharing and session hijacking.

Security

  • » Edge-level Access Control
  • » Signed URL validation at CDN edge
  • » JWT RS256 Signature Verification
  • » Rate limiting per user IP (Redis)

E. Rate Limiting & Abuse Protection

  • API Rate Limiting: Global state management via Redis to enforce sliding-window quotas across distributed nodes.
  • Preview Control: Anti-abuse system utilizing browser fingerprinting and hard daily limits on 'Free Preview' content access.
  • Wallet Integrity: Prevention of credit manipulation through server-side atomic transactions and balance validation prior to URL signing.
  • Threat Detection: IP-based behavioral analysis to identify and block bot traffic and automated scraping attempts in real-time.
